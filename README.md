## Project Overview

The Student Performance Tracker is a Machine Learning project developed to analyze and predict student academic performance based on study habits and academic factors. The project uses Linear Regression to predict student marks using features such as study hours, attendance percentage, and assignment completion.

The system also provides performance evaluation metrics, graphical visualizations, and student performance categorization to help monitor academic progress effectively.

---

## Objectives

* Track student academic performance.
* Predict student marks using Machine Learning.
* Analyze the relationship between study habits and academic results.
* Identify top-performing students.
* Generate visual reports for educational analysis.

---

## Dataset Features

The dataset contains the following attributes:

| Feature     | Description                     |
| ----------- | ------------------------------- |
| Student_ID  | Unique student identifier       |
| Study_Hours | Number of study hours           |
| Attendance  | Attendance percentage           |
| Assignments | Number of assignments completed |
| Marks       | Student examination marks       |

---

## Machine Learning Algorithm

### Linear Regression

Linear Regression is a supervised machine learning algorithm used to predict numerical values. In this project, it is used to estimate student marks based on academic performance indicators.
